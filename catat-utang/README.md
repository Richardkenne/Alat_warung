# Catat Utang Warung

Ini proyek kecil untuk bantu warung mencatat utang pelanggan.

## Cara buka proyek
1. Buka folder `catat-utang`.
2. Klik dua kali file `index.html`.
3. Situs langsung muncul di browser.

## Cara lihat preview lokal
Tidak perlu install apa-apa.
Cukup buka `index.html` seperti di atas.

## Cara pakai tool
1. Isi nama pelanggan.
2. Isi jumlah utang (IDR).
3. Tambah catatan kalau perlu.
4. Klik **Simpan Utang**.
5. Daftar utang dan total akan muncul.

Semua data disimpan di browser. Tidak perlu akun.

## Cara publish online (GitHub Pages)
1. Buat repo baru di GitHub.
2. Upload semua file di folder `catat-utang`.
3. Masuk ke **Settings** → **Pages**.
4. Pilih branch `main` dan folder root.
5. Tunggu beberapa menit, link akan jadi.

Selesai. Situs kamu sudah online.
