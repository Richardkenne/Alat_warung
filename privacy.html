Kebijakan Privasi

Terakhir diperbarui: 27 Februari 2026

1. Data yang Kami Kumpulkan

Versi Gratis: Semua data (utang, stok, keuangan) disimpan di perangkat Anda melalui browser localStorage. Kami tidak mengumpulkan, mengirim, atau menyimpan data Anda di server kami.

Versi Pro (opsional): Jika Anda memilih untuk mendaftar, kami menyimpan:

  • Nomor telepon (untuk login via OTP)
  • Nama warung (opsional)
  • Data utang, stok, dan keuangan yang Anda masukkan

Data Pro disimpan di server Supabase yang terenkripsi.

2. Bagaimana Kami Menggunakan Data

  • Menyediakan layanan pencatatan utang, stok, dan keuangan
  • Sinkronisasi data antar perangkat (hanya Pro)
  • Mengirimkan OTP untuk verifikasi login

Kami tidak menjual, menyewakan, atau membagikan data Anda kepada pihak ketiga untuk tujuan pemasaran.

3. Keamanan Data

Kami menggunakan enkripsi SSL/TLS untuk semua transfer data. Data Pro disimpan di infrastruktur Supabase dengan Row Level Security (RLS) — artinya hanya Anda yang bisa mengakses data Anda sendiri.

4. Cookies & Analytics

Situs ini tidak menggunakan cookies pelacakan. Kami mungkin menggunakan analytics sederhana (tanpa identifikasi personal) untuk memahami berapa banyak pengunjung.

5. Hak Anda

  • Akses: Anda bisa melihat semua data Anda kapan saja di aplikasi
  • Hapus: Anda bisa menghapus semua data kapan saja (versi Gratis: clear localStorage; versi Pro: hapus akun)
  • Portabilitas: Fitur export tersedia di versi Pro

6. Perubahan Kebijakan

Kami dapat memperbarui kebijakan ini. Perubahan akan dipublikasikan di halaman ini dengan tanggal terbaru.

7. Kontak

Jika ada pertanyaan tentang privasi, hubungi kami melalui WhatsApp atau email di halaman utama.
